# 📊 NovaMart Sales Analytics

## 📖 Project Overview

This is an end-to-end Data Analytics project built using SQL, Excel, Python, Power BI, and GitHub.

The project analyzes NovaMart's sales data to uncover customer behavior, product performance, revenue trends, and business insights.

---

## 🎯 Project Objectives

- Analyze sales performance
- Identify top customers
- Evaluate product performance
- Analyze payment behavior
- Build interactive dashboards
